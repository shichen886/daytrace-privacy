一、适用范围和基本原则

“日迹时光”（以下简称“本应用”）是一款本地优先的计划、记录、专注计时与每日回顾应用。本应用不提供用户注册、登录、云同步、在线社区、在线支付或云端 AI 服务。

除本政策说明的 Google ML Kit 运行指标外，本应用开发者不建立服务器接收你的计划、日记、任务、图片、OCR 结果、专注记录或备份文件。

二、无需注册账号

使用本应用不需要手机号、邮箱、QQ、第三方账号或其他登录凭证。本应用不会要求你提供密码、邮箱授权码或身份认证资料。

三、本地保存的数据

为了提供应用功能，本应用会在当前设备的应用私有存储空间中保存：

  • 分类名称、分类颜色和排序；
  • 总计划、每日任务、计划日期、任务状态及关联关系；
  • 专注计时的开始时间、暂停时间、结束时间、目标和结果；
  • 手动记录、计时成果记录、分类、关联任务和专注时长；
  • 每日统计与本地生成的每日回顾；
  • 当前主题和已同意的隐私政策版本号。

计划、任务、记录和专注数据写入应用私有的 SQLite 数据库。主题和隐私确认状态保存在应用私有文件中。这些数据用于在本机展示、搜索、统计、恢复计时状态和生成回顾。

四、图片选择与本地 OCR

只有当你主动点击“离线截图识别”并在 Android 系统文件选择器中选择图片时，本应用才会读取该图片。本应用使用 Google ML Kit 中文文字识别组件在设备上处理图片，并将识别文字显示给你确认。

根据 Google ML Kit 官方说明，OCR 输入图片、从图片识别出的文字以及识别输出不会发送到 Google 服务器。本应用开发者也不会接收这些内容。

Google ML Kit SDK 可能不定期联系 Google 服务器以获取修复、模型或硬件兼容信息，并可能发送设备信息、应用信息、非用户身份用途的安装标识、性能指标、API 配置、输入输出大小、事件类型和错误码，用于诊断、使用情况分析、维护和防止滥用。相关指标通过 HTTPS 加密传输。详情见 ML Kit 条款与隐私说明 和 Android 数据披露说明。

五、本地每日总结与分类建议

“离线概括与分类”使用应用内规则，根据当前设备中的任务数量、完成情况、专注时长、记录文字和本地关键词生成每日总结或分类建议。该功能不调用云端大模型，也不把记录内容发送到服务器。

六、备份与恢复

当你主动导出备份时，应用会生成 JSON 文件，其中可能包含分类、总计划、任务、专注记录、成果记录和每日回顾等本地数据。Android 系统文件选择器由你决定文件保存位置。

当你主动恢复备份时，应用只读取你通过系统文件选择器选中的 JSON 文件。恢复会覆盖当前应用数据库中的相应数据，应用会在操作前提示确认。

备份文件由你自行保管。请勿将包含私人内容的备份文件交给不受信任的人。如果你选择由网盘或其他文档提供方管理的位置，该提供方对文件的处理受其自身隐私政策约束。

七、权限与系统能力

系统文件选择器

本应用通过 Android Storage Access Framework 的 ACTION_OPEN_DOCUMENT 和 ACTION_CREATE_DOCUMENT 让你选择 OCR 图片、备份文件或保存位置。本应用只获得你主动选择的单个文件或目标位置的访问能力，不申请读取整个相册或全部文件的权限。

网络

本应用没有开发者运营的网络接口，也不上传用户内容。Flutter 调试版本为了热重载和开发工具通信会包含网络权限。Google ML Kit 可能按上一节所述传输 SDK 运行指标。正式安装包的最终权限以发布版本的 Android Manifest 为准。

通知、定位、相机和麦克风

当前版本没有通知或提醒功能，不申请通知、定位、相机、麦克风、通讯录、短信或通话记录权限。OCR 图片来自你主动操作的系统文件选择器，不直接调用相机。

八、第三方组件

当前版本使用 Flutter、sqflite、Google ML Kit 中文文字识别，以及 Android 系统文件选择器。各组件的用途、处理数据和联网情况见第三方 SDK 与系统组件清单。

当前版本没有接入广告 SDK、第三方用户统计 SDK、崩溃上报平台、社交登录、支付 SDK、云数据库或云端 AI SDK，也没有使用 shared_preferences、image_picker、file_picker、path_provider 或 package_info_plus。

九、数据删除和卸载

你可以在本应用提供的编辑或更多操作中删除具体任务、记录等内容，也可以在“我的”中使用双重确认清空全部本地数据。你还可以在 Android 系统设置中清除本应用的存储数据，或卸载本应用。

卸载或清除应用数据会删除应用私有数据库、主题选择和隐私确认状态。你主动导出到应用外部位置的 JSON 备份不会随卸载自动删除，需要由你在对应文件位置自行删除。

本应用已关闭 Android 自动云备份，减少应用私有数据被系统备份到云端的可能。不同设备厂商的系统迁移行为可能存在差异，请同时检查设备的备份与迁移设置。

十、数据安全

本应用使用 Android 应用私有目录保存数据库和设置，并避免将用户内容传输给本应用开发者。请为设备设置锁屏保护，并妥善保管你主动导出的备份文件。任何本地存储都无法保证绝对安全，设备取得 Root 权限、系统漏洞或他人获得设备控制权时可能增加数据风险。

十一、未成年人使用

本应用不以收集未成年人个人信息为目的，也不提供社交、支付或定向广告功能。未成年人应在监护人指导下使用，不要在记录或备份中填写不必要的身份证号、住址、联系方式等敏感信息。

十二、政策更新

功能、依赖或数据处理方式发生重要变化时，本政策会更新日期和隐私政策版本。应用会在需要时重新显示隐私确认，用户可以先查看更新后的政策，再决定是否同意继续使用。

十三、联系我们

如对本政策或本应用的数据处理有疑问，可联系 3288073775@qq.com（可长按复制）。请不要发送日记全文、备份文件、密码或其他不必要的敏感信息。
